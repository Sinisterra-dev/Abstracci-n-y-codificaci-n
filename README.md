**Autor:** Hernando Arbey Robles Puentes  
**Lenguaje:** Python 3 con Tkinter (interfaz gráfica)

---

### Descripción

Aplicación de escritorio que gestiona el pago de nómina de empleados de la empresa **Constructora Mejor**. Permite registrar datos de empleados, calcular el valor total a pagar según cargo y días laborados, y generar un reporte.

### Archivos

| Archivo | Descripción |
|---|---|
| `nomina_app.py` | Aplicación principal (login + registro + reporte) |
| `tabla_abstraccion.md` | Tabla de abstracción – Fase 2 (Anexo 2) |

### Cómo ejecutar

```bash
python nomina_app.py
```

> Requiere Python 3.9+ con Tkinter instalado.
